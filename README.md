# Javari FreeCodeCamp Scraper

Scrapes FreeCodeCamp tutorials and articles for practical coding knowledge.

## Coverage

10 key topics:
- JavaScript
- React
- Python
- Web Development
- Programming
- Data Science
- Machine Learning
- Node.js
- TypeScript
- CSS

## Schedule

- **Frequency**: Daily (0 3 * * *)
- **Time**: 3am UTC
- **Duration**: ~20-30 minutes
- **Volume**: ~200 articles per run (20 per topic)

## Features

- ✅ Latest tutorials and guides
- ✅ Full article content extraction
- ✅ Author attribution
- ✅ Topic categorization
- ✅ Smart rate limiting (1s between requests)

## Why FreeCodeCamp?

FCC provides:
- Beginner-friendly explanations
- Step-by-step tutorials
- Real-world examples
- Community-validated content

Perfect for helping Javari explain concepts clearly to users at all skill levels.
